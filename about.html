Code Stack

How We Built This

Conduit is built entirely with vanilla HTML5, CSS3, and JavaScript (ES6+), without reliance on any external UI framework.

We chose this approach to maximize performance, minimize load times, and ensure the site works reliably across all modern browsers without requiring a build pipeline or Node.js toolchain. Every line of markup, style, and logic was written directly by our team.

Pages are connected through a custom-built single-page application (SPA) architecture using the browser-native Navigation API (window.navigation.addEventListener('navigate', ...)), which intercepts internal page transitions, fetches destination HTML, and surgically swaps only the <main> content — delivering a fast, app-like experience with smooth View Transition API animations between pages.

Typography is handled through Google Fonts, pairing Fraunces — an optical-size variable serif with strong editorial character — as the display font with DM Sans for clean, readable body copy. The design system is built entirely on CSS custom properties for consistent theming, spacing, and color across every page.

Interactive maps are powered by Leaflet.js (v1.9.4) with OpenStreetMap tiles via the open tile CDN — no paid map API required. Resource pins are dynamically rendered from a local resources.json data file, color-coded by category, and clickable with custom popups.

Donation processing is integrated with Stripe for secure, PCI-compliant payment handling. User authentication, profile data, community matching results, and saved resources are managed entirely client-side using the localStorage API — no backend server or database is required, keeping the site fully static and free to host.

The site follows WCAG 2.1 accessibility guidelines: semantic HTML5 landmarks, descriptive ARIA labels, full keyboard navigability, and color contrast meeting at minimum AA standards for all text content. A skip-to-content link is present on every page for screen reader users.
